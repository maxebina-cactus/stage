# Guia de Estilo (Figma)

## 1. Cores Principais (Baseadas no Tailwind)
- Primary: Emerald (usado para botões principais, links ativos)
- Gray: Slate (usado para fundos, bordas e textos neutros)
- Success: Green-500
- Danger: Red-500

## 2. Tipografia
- Títulos (h1, h2): Fonte em negrito (font-bold), cor gray-900.
- Textos base: text-sm ou text-base, cor gray-600.

## 3. Regras de Componentes
- Botões: Devem sempre usar a prop `size="md"` e `color="primary"`.
- Cards: Devem ter sombra leve (`shadow-sm`) e bordas arredondadas (`rounded-lg`).
- Sidebar: Fica fixada na esquerda, fundo branco, com largura de 250px (w-64).
